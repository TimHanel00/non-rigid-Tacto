Welcome to the Open Simulation Pipeline for Deforming Organs!
=============================================================

Setting up many simulations to train machine learning algorithms is a lot of work.
This project aims to aid researchers by providing commonly used functionality, example pipelines and the building blocks to set up their own pipelines.
